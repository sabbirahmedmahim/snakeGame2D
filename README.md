# Snake Game (C++ Console)

A simple console-based Snake game written in **C++** using basic game-loop logic, real-time keyboard input, and array-based movement.

This project is meant for **learning**, not fancy graphics.

---

## Controls

| Key | Action |
|---|---|
| W | Move Up |
| A | Move Left |
| S | Move Down |
| D | Move Right |
| X | Quit Game |
